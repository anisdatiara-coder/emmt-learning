1. Garis

Untuk menggambar garis, gunakan perintah LINE. Anda dapat mengklik dua titik pada area gambar untuk membuat garis.

2. Polyline

Pada polyline, gunakan perintah PLINE untuk menggambar garis yang terhubung secara bertahap.

3. Offset

Perintah OFFSET digunakan untuk membuat garis sejajar dengan jarak yang ditentukan.

4. Lingkaran

Gunakan perintah CIRCLE dan klik pada titik pusat serta tentukan radiusnya.

5. Persegi Panjang

Untuk menggambar persegi panjang, gunakan perintah RECTANGLE dan klik dua sudut berlawanan.

6. Dimensi

Gunakan perintah DIMLINEAR untuk menambahkan dimensi garis. Pilih dua titik untuk membedakan jarak.

7. Cermin

Dengan perintah MIRROR, Anda dapat mencerminkan objek yang ada.

8. Potong

Gunakan perintah TRIM untuk memotong objek sesuai dengan garis batas yang ditentukan.

9. Perpanjang

Gunakan perintah EXTEND untuk memperpanjang objek hingga batas yang ditentukan.

10. Skala

Perintah SCALE mengubah ukuran objek sesuai dengan faktor yang ditentukan.

11. Pindah

Gunakan perintah MOVE untuk memindahkan objek ke lokasi baru.

12. Putar

Perintah ROTATE digunakan untuk memutar objek pada sudut tertentu.

13. Operasi Blok

Gunakan perintah BLOCK untuk mengelompokkan objek.

14. Array

Perintah ARRAY digunakan untuk membuat salinan objek dalam pola tertentu.

15. Layer

Atur dan kelola layer dengan perintah LAYER.

16. Fitur Lainnya

AutoCAD memiliki banyak fitur canggih lainnya, termasuk Annotation dan Layouts.
